Internship Description

TechZim Solutions is seeking a talented and motivated Software Engineering Intern to join our dynamic development team in Harare. This internship offers an exceptional opportunity for a third-year computer science student to gain hands-on experience in real-world software development projects while working alongside experienced professionals.

As a Software Engineering Intern, you will be involved in the full software development lifecycle, from requirements gathering and design to implementation, testing, and deployment. You will work on challenging projects using modern JavaScript frameworks and technologies, contributing to our suite of web applications that serve clients across Zimbabwe and beyond.

This is a paid internship position with the potential for full-time employment upon successful completion and graduation.

Key Responsibilities:

  • Develop and maintain web applications using React, Node.js, and MongoDB
  • Collaborate with senior developers to design, implement, and test new features
  • Write clean, efficient, and well-documented code
  • Participate in code reviews and contribute to improving development practices
  • Debug and fix issues in existing applications
  • Learn and implement best practices for web development and security
  • Assist in API development and integration
  • Contribute to technical documentation

Requirements:

  • Currently enrolled in the third year of a Computer Science, Software Engineering, or related program at Harare Institute of Technology
  • Strong knowledge of JavaScript, HTML, and CSS
  • Familiarity with React.js and Node.js is highly desirable
  • Basic understanding of database systems (SQL or NoSQL)
  • Good problem-solving skills and attention to detail
  • Ability to work in a team environment
  • Strong communication skills in English, both written and verbal
  • Eager to learn and adapt to new technologies
  • Previous project experience (academic or personal) is a plus

Benefits:

  • Competitive monthly stipend of $500
  • Flexible working hours compatible with academic schedule
  • Mentorship from experienced software engineers
  • Exposure to real-world projects and industry best practices
  • Modern office environment with necessary equipment provided
  • Weekly team lunches and social events
  • Networking opportunities with industry professionals
  • Potential for full-time employment after graduation

How to Apply

To apply for this internship position, please submit your application through our online portal. The application process involves the following steps:

  1. Submit your application with your CV/resume and cover letter
  2. Applications will be reviewed within 5-7 business days
  3. Shortlisted candidates will be invited for a technical interview
  4. Final selection will be made based on interview performance and skills match

Required Documents

  • Updated CV/resume
  • Cover letter explaining your interest in the position
  • Academic transcript (unofficial is acceptable)
  • Portfolio or links to previous projects (if available)
  • Reference letter from a professor or previous employer (optional)
